# Architecture Overview

This project represents a simplified cloud-native platform stack for deploying AI-ready containerized workloads.

## Components

- **Terraform** provisions cloud infrastructure
- **Kubernetes** runs containerized inference workloads
- **Prometheus** provides monitoring configuration
- **Shell scripts** automate deployments
- **GitHub Actions** validates and deploys infrastructure changes

## Goals

- Demonstrate platform engineering structure
- Show infrastructure-as-code practices
- Highlight observability and automation patterns
- Provide a recruiter-friendly DevOps portfolio example
